# Análisis Financiero Estimado del Proyecto "Poli 2.0"

## Advertencia
Este documento proporciona estimaciones basadas en estándares de la industria del software en Colombia y en la información del proyecto. Las cifras no constituyen una valoración financiera formal y deben ser utilizadas únicamente como una guía de referencia.

---

## 1. Valoración Estimada del Proyecto (Activo de Software)

Esta valoración se basa en el esfuerzo de desarrollo estimado (horas) y tarifas promedio del mercado colombiano para un equipo de desarrollo calificado.

### 1.1. Desglose de Horas de Desarrollo Estimadas

| Área de Desarrollo                                    | Horas Estimadas | Descripción                                                                                   |
| ----------------------------------------------------- | --------------- | --------------------------------------------------------------------------------------------- |
| **Frontend (UI/UX y Lógica de Cliente)**              | **450 horas**   | Diseño de interfaces, implementación de componentes, vistas para cada rol, responsividad.       |
| **Backend (APIs, Lógica de Negocio, Base de Datos)** | **400 horas**   | Creación de APIs, lógica de registro/login, gestión de usuarios, horarios, notas, etc.        |
| **Integración de IA (Genkit)**                        | **80 horas**    | Desarrollo de flujos para correos, sugerencias y otras funcionalidades de IA.                 |
| **Gestión, Pruebas y Despliegue (QA & DevOps)**       | **150 horas**   | Planificación, pruebas funcionales, configuración de despliegue en Firebase.                  |
| **Total Estimado**                                    | **1,080 horas** |                                                                                               |

### 1.2. Tarifa por Hora (Promedio en Colombia)

*   **Tarifa Junior-Mid:** COP $60,000 - $90,000 / hora
*   **Tarifa Senior/Especialista:** COP $100,000 - $150,000 / hora
*   **Tarifa Promedio de Equipo Mixto:** **COP $80,000 - $120,000 / hora**

### 1.3. Cálculo de la Valoración

*   **Escenario Conservador:** 1,080 horas * $80,000/hora = **$86,400,000 COP**
*   **Escenario Optimista:** 1,080 horas * $120,000/hora = **$129,600,000 COP**

> **Conclusión de Valoración:** El valor intrínseco del software, basado en su construcción, se estima entre **ochenta y seis y ciento treinta millones de pesos colombianos**. Este valor no incluye potencial de mercado, marca o ingresos futuros, que podrían aumentarlo significativamente.

### 3.2. Escenario de Ganancias Hipotético

**Supuestos:**
*   El software se licencia a una institución educativa.
*   **Ingreso Bruto Anual (Licencia):** **COP $150,000,000**
*   **Costo de Mantenimiento Anual (Rango Medio):** $3,000,000/mes * 12 = **COP $36,000,000**

*   Ganancia Neta Anual = Ingresos Brutos - Costos de Mantenimiento
